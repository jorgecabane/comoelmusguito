# Open Graph Images (Para Redes Sociales)

## ¿Qué son?

Imágenes que aparecen cuando compartes la web en:
- Facebook
- Twitter/X
- WhatsApp
- LinkedIn
- etc.

## Especificaciones CRÍTICAS

**Tamaño:** 1200x630 exacto (ratio 1.91:1)
**Peso:** <200KB
**Formato:** JPG

## Archivos Necesarios

### 1. Home
```
home.jpg
- Preview al compartir página principal
- Debe tener: logo + terrario + tagline
```

### 2. Default
```
default.jpg
- Fallback para páginas sin OG específico
- Mismo contenido que home
```

### 3. Terrarios (Opcional)
```
terrarios.jpg
- Para página de catálogo de terrarios
```

### 4. Cursos (Opcional)
```
cursos.jpg
- Para página de cursos online
```
