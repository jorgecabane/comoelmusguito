# Terrarios Productos

## Naming Convention

```
terrario-{nombre}-{número}.jpg

Ejemplo:
terrario-bosque-nublado-01.jpg
terrario-bosque-nublado-02.jpg
terrario-desertico-01.jpg
```

## Especificaciones

- Tamaño: 2000x2000 (cuadrado 1:1)
- Peso: <300KB por imagen
- Formato: JPG optimizado
- Mínimo: 2 fotos por terrario

## Tips Fotográficos

**Ángulos recomendados:**
1. Frontal 45° (principal)
2. Close-up de plantas/musgo
3. Vista lateral mostrando capas
4. Top-down (opcional)

**Iluminación:**
- Natural indirecta
- Fondo neutro (blanco, crema, madera)
- Evitar reflejos en vidrio
